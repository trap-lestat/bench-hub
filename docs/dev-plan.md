# 开发任务清单（实际开发）

## 核心目标
- 提供压测管理后台：脚本管理、任务运行、报告查看
- 后端可触发 Locust 运行并读取报告结果
- 脚本可新增、修改、导入、删除
- 任务可配置用户数、时长、压测参数并可追踪状态

## 最小可行版本（MVP）
- 脚本管理：列表、新增、编辑、删除、导入
- 任务管理：新建任务（用户数、spawn rate、时长）、运行、停止
- 报告查看：HTML/CSV 列表与详情预览
- 后端执行：调用 Locust headless 并落盘报告

## 模块拆分
- 脚本库模块：脚本元数据、版本、存储、导入/导出
- 任务调度模块：任务创建、运行、状态跟踪
- 报告模块：报告生成、存储、展示
- 权限与审计：操作记录、用户权限控制
- 前端管理台：脚本/任务/报告的页面与交互
